# 形態素解析＋Bayesを利用した文章からの予測アルゴリズム

起動
`cd app`
`npm start`

形態素解析にはatilika社によるkuromojiをTakuya Asano氏がjavascript化したkuromoji.jsを利用
bayesモジュールを利用。
